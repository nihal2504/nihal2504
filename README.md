## 👋 About Me

I'm a **Junior Software Engineer at NITMiner Technologies**, working on **TrustInn** — an IBITF-sponsored blockchain security research platform at NIT Warangal.

My work sits at the intersection of **formal verification**, **smart contract security**, and **applied machine learning**. I build tools that find vulnerabilities in code so humans don't have to hunt for them manually.

- 🔐 **Day job:** BMC + DSE pipelines for Solidity smart contracts and C/C++ codebases (CBMC, KLEE, solc-bmc)
- 🤖 **ML focus:** Privacy-preserving systems — Federated Learning + Differential Privacy (Opacus)
- 🦀 **Currently learning:** Rust for systems-level formal verification (Kani verifier)
- 🎓 **Background:** B.Tech Computer Science Engineering, KITS Warangal (2025)
